🎯 Spray Charts

Importer un spray chart depuis GameChanger

Comment récupérer le SVG dans GameChanger :
1. Allez sur la page du joueur dans GameChanger Web (app.gc.com)
2. Cliquez sur le spray chart pour l’afficher
3. Clic droit › Inspecter › trouvez le bloc <svg> du spray chart
4. Clic droit sur <svg> › Copy › Copy outerHTML
5. Collez dans le champ ci-dessus
